# Play 24 Game

Given four numbers, find a way to combine them using +, -, *, / to make exactly 24.

## Examples

* "Solve 24 game with 1, 2, 3, 4"
* "Can you make 24 from 8, 3, 8, 3?"
* "Play 24 with 5, 5, 5, 1"

## Instructions

Call the `run_js` tool with the following exact parameters:

- data: A JSON string with the following fields:
  - numbers: An array of exactly 4 numbers. Required.
